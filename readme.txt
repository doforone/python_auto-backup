这两个小程序：
	001：把指定的目录及子目录下的文件连同目录备份到另一（硬盘）目录下，比如把“D:\”备份到“E:\DDD\”。
	002：执行完001后，执行002，002会把备份目录（比如“E:\DDD\”）及子目录中的文件跟源目录（比如“D:\”）中的文件去比对，如果源文件不存在了就删除，原因是源目录中的文件已经删除了，执行001的时候只是正向进行了备份，002执行的时候才进行反向比对，进行删除操作。

两个程序可以合并成一个，如果还要更快速度，可以把部分屏幕输出注释掉。

如果需要实时备份，也即一个文件或目录发生了变化（如创建、修改、删除、重命名）就立刻同步备份，这个需要用操作系统的的API事件监控文件变化，然后用回调函数执行备份，如需此功能，请跟作者联系。
